# Лабораторная работа №3

---

## Описание заданий

1. **Вычисление значений кусочной функции** - Вывод таблицы значений.

2. **Тест Миллера + решето Эратосфена** — генерация простых чисел заданной разрядности с подсчётом отвергнутых кандидатов.

3. **Математическая сумма** — вычисление суммы бесконечного ряда, приведение к несократимой дроби.

4. **Увлекательная игра (вариант 2)** — пошаговое удаление чисел с запретом на повтор хода противника, выбор максимальной суммы.

5. **Остывание кофе** — моделирование по закону Ньютона, линейная регрессия, коэффициент корреляции.
